# Assets Directory

## System Tray Icon

You need to add a `icon.ico` file to this directory for the system tray icon.

### Requirements:
- **Format**: .ico (Windows icon format)
- **Size**: 16x16 or 32x32 pixels recommended
- **File name**: `icon.ico` (exact name)

### How to Create:
1. Create or find a printer icon image
2. Convert to .ico format using an online tool like:
   - https://convertio.co/png-ico/
   - https://www.icoconverter.com/
3. Save as `icon.ico` in this directory

### Temporary Solution:
The application will work without an icon, but the tray icon will appear blank.
For production use, you should add a proper icon file.

### Suggested Icon Style:
- Simple printer icon
- High contrast for visibility in system tray
- Recognizable at small sizes (16x16)
